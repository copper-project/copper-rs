# cu-udp-inject

## Overview

A Simple pcap UDP injector to test network sensors with Copper (cu29 crate)

## Interface

Provides test/injection helpers for validating Copper data flow.

## Configuration

Set test endpoints and behavior through task `config` entries.

## Usage

Use in test/demo graphs to inject inputs and assert end-to-end behavior.

## Compatibility

Intended for testing environments; feature support follows crate dependencies.
