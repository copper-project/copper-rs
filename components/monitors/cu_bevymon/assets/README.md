# Bundled Fonts

`cu_bevymon` bundles a small JetBrains Mono Nerd Font Mono set for the Bevy windowed Ratatui backend:

- `JetBrainsMonoNerdFontMono-Light.ttf`
- `JetBrainsMonoNerdFontMono-SemiBold.ttf`
- `JetBrainsMonoNerdFontMono-LightItalic.ttf`

These files were copied from the local Arch Linux `ttf-jetbrains-mono-nerd` package so the Bevy monitor does not depend on host font discovery at runtime.

The upstream font license is included as [OFL.txt](/home/gbin/projects/copper/copper-rs.bevy_monitor/components/monitors/cu_bevymon/assets/fonts/OFL.txt).
