# cu_zenoh

## Overview

This demonstrates how to use Copper to interface with the zenoh communication middleware.

## Prerequisites

- Rust stable toolchain
- Any hardware or services referenced by this example

## Run

```bash
cargo run -p cu-zenoh
```

## Expected Output

Expect successful startup logs and normal task-loop execution without panics.
