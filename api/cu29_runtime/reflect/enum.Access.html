Enum Access 

pub enum Access<'a> {
    Field(Cow<'a, str>),
    FieldIndex(usize),
    TupleIndex(usize),
    ListIndex(usize),
}
Expand description

A singular element access within a path. Multiple accesses can be combined into a ParsedPath.

Can be applied to a dyn Reflect to get a reference to the targeted element.
