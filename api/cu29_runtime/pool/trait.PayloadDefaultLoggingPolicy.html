Expand description

Default arm of the autoref-specialization pattern used by the unified-log encoder.

Blanket-impl’d for every type, so any payload that doesn’t define its own inherent payload_should_log method inherits this default-true implementation. Types like CuHandle (and composite payloads wrapping one) provide an inherent method with the same name, which wins method resolution because inherent methods are tried before trait methods at the same candidate self-type.
