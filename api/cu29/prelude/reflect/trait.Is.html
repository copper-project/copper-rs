Checks if the current type “is” another type, using a TypeId equality comparison.

Required Methods§

fn is<T>() -> bool
where T: Any,

Checks if the current type “is” another type, using a TypeId equality comparison. This is most useful in the context of generic logic.

fn greet_if_u32<T: Any>() {
    if T::is::<u32>() {
        println!("Hello");
    }
}
// this will print "Hello"
greet_if_u32::<u32>();
// this will not print "Hello"
greet_if_u32::<String>();
assert!(u32::is::<u32>());
assert!(!usize::is::<u32>());
